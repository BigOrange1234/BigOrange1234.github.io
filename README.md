# 双色球AI智能选号助手 V23-稳定增强版

这是双色球AI智能选号助手的V23稳定增强版本，提供了完整的历史数据分析、AI预测和智能选号功能。

## 功能特点

- 📊 **历史数据查询**：支持查看2025年第1-16期的完整开奖数据
- 📈 **数据分析图表**：提供出现频率、分布分析、趋势分析和相关性分析
- 🤖 **AI预测分析**：基于历史数据进行智能预测，推荐可能的中奖号码
- 🎯 **智能选号生成**：支持随机选号、热门号码和AI推荐三种模式
- ⚙️ **高级设置**：可配置排除历史重复、平衡奇偶比例、限制连号数量等选项
- 💾 **数据导出**：支持将历史数据导出为CSV格式

## 快速开始

### 方法一：使用内置服务器（推荐）

1. 确保已安装Node.js（版本14.0.0或更高）
2. 打开命令行工具，进入项目目录
3. 运行以下命令启动服务器：
   ```bash
   npm start
   ```
4. 打开浏览器，访问 http://localhost:3003

### 方法二：直接打开HTML文件

1. 在文件管理器中找到 `index.html` 文件
2. 双击打开，或右键选择浏览器打开

## 使用说明

### 历史数据查询

1. 在"历史开奖数据"区域，您可以查看历史开奖记录
2. 使用筛选下拉框选择不同的时间范围（全部、2025年、最近10期等）
3. 点击"查询"按钮应用筛选条件
4. 使用分页按钮浏览更多历史数据

### 数据分析

1. 在"数据分析"区域，选择图表类型（出现频率、分布分析等）
2. 选择时间范围（全部、本年度、近半年等）
3. 点击"更新"按钮生成图表
4. 鼠标悬停在图表上可查看详细数据

### AI预测分析

1. 在"AI预测分析"区域，点击"开始分析"按钮
2. 系统将自动分析历史数据，生成预测结果
3. 结果包括热门号码、冷门号码和AI推荐号码

### 智能选号

1. 在"智能选号生成器"区域，选择选号模式：
   - 随机选号：完全随机生成号码组合
   - 热门号码：基于历史高频号码生成
   - AI推荐：结合多种算法智能推荐
2. 设置选号数量（1-10注）
3. 配置高级选项（可选）
4. 点击"生成选号"按钮
5. 在"选号结果"区域查看生成的号码
6. 可点击单个选号的复制按钮复制特定选号，或点击"复制"按钮复制所有选号
